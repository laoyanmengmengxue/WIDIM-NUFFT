# WIDIM-NUFFT

Official repository for the paper:

**Direct Local Rotation-Rate Measurement for Particle Image Velocimetry via Spectral Angular-Shift Estimation**

## Repository layout

- `paper/`: main manuscript, supplement, bibliography, and the figures used in the paper
- `code/`: core algorithms, bridge-benchmark code, figure-generation scripts, and real-data run scripts
- `data/`: example real-data and bridge-case inputs
- `results/`: representative synthetic, bridge, and structure-validation outputs

## Notes

- The large real-data batch outputs can be added later under `results/real_vortex_all_p5/`.
- Additional bridge-benchmark batch outputs can be added later under `results/bridge_benchmark/`.
- Real-data source links are summarized in `real_data_sources.txt`.
